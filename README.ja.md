# monoimgmaker

JPGおよびPNG画像を、電子ペーパーディスプレイ用にフォーマットされたモノクロのC言語ソースファイルに変換するWebツールです。

## デモ

**https://code4fukui.github.io/monoimgmaker/**

Webインターフェースには、画像ファイルをドラッグ＆ドロップするためのシンプルなエリアが用意されています。変換処理はブラウザ内で行われ、生成されたC言語ソースファイルが自動的にダウンロードされます。

## 機能

- **Webベース:** インストール不要で、すべてブラウザ上で動作します。
- **ドラッグ＆ドロップ:** シンプルで高速なユーザーインターフェース。
- **一般的なフォーマットに対応:** JPGおよびPNGファイルを変換します。
- **電子ペーパー対応:** 7.5インチ電子ペーパーディスプレイのライブラリで一般的なフォーマットである `const unsigned char gImage_7in5_V2[]` 配列を生成します。
- **モノクロ変換:** グリーンチャンネルの固定しきい値（`G < 100`）を使用して、画像を1ビットの白黒データに変換します。

## 使い方

1. ブラウザで[デモページ](https://code4fukui.github.io/monoimgmaker/)を開きます。
2. 指定されたエリアにJPGまたはPNG画像ファイルをドラッグ＆ドロップします。
3. 変換されたC言語ソースファイル `ImageData.c` が自動的にダウンロードされます。

**重要:** このツールは、サイズが48,000バイトにハードコードされたC言語の配列を生成します。入力画像は、この正確なサイズになる寸法である必要があります（例: 800x480ピクセル、`800 * 480 / 8 = 48000` のため）。

## 出力例

生成される `ImageData.c` ファイルには以下の構造が含まれます:

```c
#include "ImageData.h"
const unsigned char gImage_7in5_V2[48000] = {
0x00, 0x01, /* ... more byte data ... */
};
```
