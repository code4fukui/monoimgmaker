# monoimgmaker

1 枚の画像ファイルをモノクロのC言語ソースファイルに変換するウェブアプリです。

## デモ
https://code4fukui.github.io/monoimgmaker/

## 機能
- JPG/PNG 形式の画像ファイルを受け取ります
- 画像をモノクロに変換し、C 言語のソースコードとしてダウンロードできます

## 使い方
1. ブラウザで上記のデモページを開きます
2. JPG/PNG 形式の画像ファイルをドロップします
3. 変換が完了すると、`ImageData.c` というファイル名でダウンロードが始まります
