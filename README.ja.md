# monoimgmaker

1枚の画像ファイルをモノクロのC言語ソースファイルに変換するウェブアプリです。

## デモ
https://code4fukui.github.io/monoimgmaker/

## 機能
- JPG/PNG形式の画像ファイルを受け取り
- 画像をモノクロに変換し、C言語のソースコードとしてダウンロードできます

## 使い方
1. ブラウザで上記のデモページを開きます
2. JPG/PNG形式の画像ファイルをドロップします
3. 変換が完了すると、`ImageData.c`というファイル名でダウンロードが始まります
