# monoimgmaker

Converts image files to monochrome C language source files.

## Demo
https://code4fukui.github.io/monoimgmaker/

## Features
- Converts JPG and PNG image files to monochrome C source files
- Provides a simple web interface to drag and drop files

## Usage
1. Drag and drop image files (JPG or PNG) onto the web page.
2. The converted C source file will be downloaded automatically.
